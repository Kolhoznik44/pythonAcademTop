Задание 1. Алгоритм управления автомобилем 
Садимся в автомобиль.
Включаем зажигание.
Проверяем наличие топлива.
Если топлива нет, заправляемся.
Включаем передачу.
Нажимаем педаль газа и рулём управляем движением.
Если возникает препятствие, тормозим или объезжаем.
По достижении точки назначения — тормозим и выключаем зажигание.
Выходим из автомобиля.
Блок-схема (текстовая):
[Старт] → Сесть в автомобиль → Включить зажигание → Проверить топливо
→ [Топливо есть?] → Да → Следующий шаг
→ Нет → Заправиться → Включить передачу
→ Управлять движением
→ [Препятствие?] → Да → Тормозить/Объехать
→ Нет → Двигаться
→ [Достигли цели?] → Нет → Управлять дальше
→ Да → Остановить автомобиль → Выключить зажигание → Выйти [Стоп]

Задание 2. Алгоритм покупки велосипедных шин (словесно)
Определяем, какие шины нужны.
Проверяем наличие денег.
Выбираем магазин.
Приходим в магазин.
Ищем нужные шины.
Проверяем цену и качество.
Покупаем, если всё устраивает; если нет — идём в другой магазин.
Покупка завершена.
Блок-схема (текстовая):
[Старт] → Определить нужные шины → Проверить деньги
→ [Хватит денег?] → Нет → Скопить деньги [Стоп]
→ Да → Выбрать магазин → Прийти в магазин
→ Найти шины → [Подходят?] → Да → Купить шины [Стоп]
→ Нет → Перейти в другой магазин → повторить

Задание 3. Алгоритм написания романа (словесно)
Формулируем идею и сюжет.
Разрабатываем план глав.
Пишем черновик первой главы.
Повторяем для остальных глав.
Редактируем текст.
Читаем и исправляем ошибки.
Завершаем роман.
Блок-схема (текстовая):
[Старт] → Сформулировать идею/сюжет → Составить план
→ [Следующая глава есть?] → Да → Написать черновик главы → Повторить
→ Нет → Редактировать текст → Исправить ошибки → Завершить роман [Стоп]

Задание 4. Алгоритм установки дверей (словесно)
Измеряем дверной проём.
Покупаем подходящую дверь.
Подготавливаем инструменты.
Снимаем старую дверь (если есть).
Устанавливаем новую коробку (если надо).
Навешиваем дверь на петли.
Проверяем, как дверь открывается и закрывается.
При необходимости регулируем.
Зачищаем, убираем инструменты.
Блок-схема (текстовая):
[Старт] → Измерить проём → Купить дверь
→ Подготовить инструменты → Снять старую дверь
→ Установить коробку → Навесить дверь
→ Проверить функционирование
→ [Нужно регулировать?] → Да → Отрегулировать
→ Нет → Зачистить место → Убрать инструменты [Стоп]

Задание 5. Алгоритм управления скейтом (словесно)
Надеть защиту.
Встать на скейт.
Оттолкнуться ногой для движения.
Балансировать телом.
Поворачивать переносом веса или поворотом доски.
Если нужно остановиться — тормозить ногой, или прыгнуть.
По необходимости повторять.
Блок-схема (текстовая):
[Старт] → Надеть защиту → Встать на скейт
→ Оттолкнуться → Балансировать
→ [Нужно повернуть?] → Да → Повернуть
→ Нет → [Нужно остановиться?] → Да → Остановиться/Сойти [Стоп]
→ Нет → Продолжать кататься

Задание 6. Алгоритм мытья посуды (словесно)
Собрать грязную посуду.
Убрать остатки еды.
Намылить губку моющим средством.
Промыть каждую тарелку/стакан/вилку.
Прополоскать под водой.
Поставить на сушилку.
После окончания — вытереть раковину.
Блок-схема (текстовая):
[Старт] → Собрать посуду → Очистить от еды
→ Намылить губку
→ [Есть ещё грязная посуда?] → Да → Вымыть → Прополоскать → Поставить
→ Нет → Вытереть раковину [Стоп]
